About MaxTac

Description

MaxTac is a project that consists of three sub projects namely: Alexandrite, Vermillion and Shadow. Each sub projects co-exists by dedicating data to each other.


Experiencing API Issues

Alexandrite is using Wolfram Alpha API for Node.js for computation. The API was functioning perfectly until one day my console returned an error.

The API required the Node.js version to be ~0.6.10 whereas i was running Node.js v21.3.0.

I will have to recreate the entire project from scratch and hoping with Wolfram Alpha API.

This issue had delayed the project from advancing by 2 weeks as i have exams and less time to dedicate to my project.


Alexandrite

Alexandrite Magnus is a Computational Intelligence with the ability of AI. It can solve difficult to complex level mathematical inputs in just matter of time.

It can also return weather details, wikipedia results and return word meanings.


Vermillion

Scarlett Vermillion is an AI that uses Sentiment Analysis to return appropriate responses to the user and also powers Shadow's Machine Learning knowledge.

Uses custom made Natural Language Processing aka NLP as the root for AI abilities.


Shadow

Maximus Shadow is a Machine Learning Intelligence that uses Vermillion's data to feed data into the dataset. Vermillion then uses Shadow's input and returns the best appropriate response.

Uses Tensorflow.js, Limdu.js and KerasJS as Machine Learning libraries.
